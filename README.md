🎬 Proyecto GR06 – Sistema de Gestión de Películas y Series
Este repositorio contiene el proyecto para el curso de Ingeniería Web II.
Incluye:
- Backend: API REST con Node.js, Express y MongoDB.
- Frontend: Aplicación web en React para gestionar películas, series y entidades relacionadas.

---

🧩 Estructura del repositorio:

Peliculas_DB/
│
├── backend/     # API REST
│
└── frontend/    # Aplicación React
Cada carpeta contiene su propio README con instrucciones detalladas.

---

🚀 Tecnologías principales
🔹 Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- Dotenv
- Postman
🔹 Frontend
- React
- Axios
- React Router
- Bootstrap / CSS
  
--

⚙️ Cómo ejecutar el proyecto completo
1. Clonar el repositorio
   git clone https://github.com/Disbeidy/Peliculas_DB.git
   cd Peliculas_DB
2. Ejecutar el Backend
   cd backend
   npm install
   npm start
El backend quedará disponible en:
http://localhost:4000

💻 Ejecutar el Frontend
cd frontend
npm install
npm start
El frontend abrirá en:
http://localhost:3000
